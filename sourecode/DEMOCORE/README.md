## Quick Start

### Open the Prototype (zero setup)
```
Open TASKS/04_prototype.html in Chrome / Firefox / Edge
```
No server, no npm, no API keys. Just open the file.

### Run the AI/ML Simulation (Python only)
```bash
# Analyze a specific wallet
python TASKS/03_qubo_sim.py 0xd90e2f925DA726b50C4Ed8D0Fb90Ad053324F31b

# Get JSON output
python TASKS/03_qubo_sim.py 0xd90e2f925DA726b50C4Ed8D0Fb90Ad053324F31b --json

# Start HTTP API server (for advanced integration)
python TASKS/03_qubo_sim.py serve 8765
# Then: GET http://localhost:8765/analyze?wallet=0x...

# Pre-compute all demo wallets
python TASKS/03_qubo_sim.py precompute
```
